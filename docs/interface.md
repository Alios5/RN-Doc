# Interface

Discover the powerful interface of RhythmNator Editor.

## Home Screen

The Home Screen is your starting point. It allows you to manage your projects and customize your environment via the top toolbar (themes, shortcuts, fonts, languages).

[Check out Home Interface details](./interface-home.md)


## The Editor

The main workspace where the magic happens.

### 1. Audio Waveform
Located at the top, it shows the visual representation of your audio. Use it to navigate precisely.

### 2. Multi-track Timeline
The central part of the editor.
- **Grids**: Visual beat and measure lines.
- **Snapping**: Toggle grid snapping for perfect timing.
- **Zoom**: Zoom in for detailed mapping or out for a big picture view.
- **Smart Groups**: Organise your tracks and notes with powerful grouping logic.

### 3. Toolbar
Quick access to selection tools, note placement, and playback controls.

### 4. Note Actions & Export
- **Custom Actions**: Select any note to assign game-specific logic or actions in the properties panel.
- **JSON Export**: Export your charts to a clean, production-ready JSON format.
