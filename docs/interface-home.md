# Interface: Home

The Home Screen is your project hub.

## Project List
Manage your `.rne` files. You can open, rename, or delete projects directly from here.

## Theme Selector
Customize your experience with the palette icon. 
- **Amethyst**: Our signature purple theme.
- **Dark/Light**: Classic modes.
- **T3 Chat**: A special magenta-inspired theme.

## Create New Project
Enter your details and import an audio track to get started.
