# Getting Started

Ready to create your first rhythm chart? Follow this guide to set up your project and start mapping.

## Prerequisites
- A Windows computer.
- An audio file (MP3, WAV, or OGG).

## Step-by-Step
1. **Launch**: Open RhythmNator Editor.
2. **New Project**: Click on the "New Project" button.
3. **Setup**: Name your project and select your audio file.
4. **BPM**: Use the auto-detect tool or set it manually.
5. **Save**: Press `Ctrl+S` to save your project frequently.
