# Interface: Editor

The Sheet Editor is where you create the rhythms.

## Components
- **Waveform Browser**: Navigation and synchronization.
- **Timeline Lanes**: Place notes on different tracks.
- **Note Properties**: Customize individual note behaviors.

## Advanced Grid
- **Beat Lines**: Vertical lines for timing.
- **Snapping**: Lock notes to the grid for perfect accuracy.
- **Zoom**: Scalable view of the entire track.
