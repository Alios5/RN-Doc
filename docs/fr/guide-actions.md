# Guide : Actions

Définissez une logique spécifique à votre jeu avec les Actions.

## Actions de note personnalisées
Au-delà du simple placement, chaque note peut déclencher une action spécifique.
- **note_hit** : Logique de coup standard.
- **special_power** : Déclenche un multiplicateur ou un effet visuel.
- **camera_shake** : Fait trembler la caméra du jeu en synchronisation avec le rythme.

## Édition des actions
Sélectionnez une note et utilisez le panneau "Actions" pour attribuer ou créer de nouveaux déclencheurs.
