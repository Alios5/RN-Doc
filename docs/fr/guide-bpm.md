# Guide : BPM

Le succès dans les jeux de rythme dépend d'un BPM parfait.

## Détection automatique
RhythmNator Editor peut analyser automatiquement votre piste pour trouver le BPM moyen.

## Réglage manuel
Si la détection n'est pas parfaite, vous pouvez saisir manuellement la valeur du BPM dans le panneau latéral.

## Décalage (Offset)
Ajustez le décalage (en millisecondes) pour aligner parfaitement le premier temps ("1") avec la forme d'onde audio.
