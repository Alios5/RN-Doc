# Démarrage

Prêt à créer votre première partition rythmique ? Suivez ce guide pour configurer votre projet et commencer le mapping.

## Prérequis
- Un ordinateur sous Windows.
- Un fichier audio (MP3, WAV ou OGG).

## Étapes par étape
1. **Lancement** : Ouvrez RhythmNator Editor.
2. **Nouveau Projet** : Cliquez sur le bouton "Nouveau projet".
3. **Configuration** : Nommez votre projet et sélectionnez votre fichier audio.
4. **BPM** : Utilisez l'outil de détection automatique ou réglez-le manuellement.
5. **Sauvegarde** : Appuyez sur `Ctrl+S` pour sauvegarder votre projet fréquemment.
