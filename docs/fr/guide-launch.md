# Guide : Lancement de projet

Comment démarrer et sauvegarder avec succès vos projets RhythmNator.

## Ouvrir un projet
Cliquez simplement sur une carte de projet depuis le tableau de bord de l'accueil.

## Format de fichier
Les projets sont sauvegardés au format `.rne`. Ce sont des fichiers légers basés sur JSON qui stockent les positions des notes et les métadonnées.

## Sauvegarde automatique et manuelle
Utilisez toujours `Ctrl+S` pour vous assurer que votre progression est sauvegardée. L'éditeur vous avertira si vous avez des modifications non sauvegardées avant de fermer.
