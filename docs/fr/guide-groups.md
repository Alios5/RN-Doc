# Guide : Groupes

Organisez votre chronologie avec les Groupes Intelligents (Smart Groups).

## Utilisation des groupes
Les groupes vous permettent de rassembler plusieurs pistes. C'est utile pour séparer les différentes sections d'une chanson ou différents types d'instruments (par exemple, toutes les pistes de percussion dans un seul groupe).

## Logique et Hiérarchie
- **Visibilité** : Activez ou désactivez la visibilité du groupe pour épurer votre espace de travail.
- **Actions groupées** : Appliquez des modifications à toutes les pistes d'un groupe simultanément.
- **Export** : Les groupes sont préservés dans l'export JSON pour une organisation logique dans votre jeu.
