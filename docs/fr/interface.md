# Interface

Découvrez l'interface puissante de RhythmNator Editor.

## Écran d'accueil

Lorsque vous lancez l'application pour la première fois, vous êtes accueilli par l'écran d'accueil.
- **Liste des projets** : Visualisez et gérez vos projets existants.
- **Créer un nouveau projet** : Démarrez un nouveau projet en spécifiant un nom et un fichier audio.
- **Thèmes** : Cliquez sur l'icône de palette pour basculer entre les 6 thèmes intégrés.

## L'Éditeur

L'espace de travail principal où la magie opère.

### 1. Forme d'onde audio
Située en haut, elle montre la représentation visuelle de votre audio. Utilisez-la pour naviguer avec précision.

### 2. Chronologie multi-pistes
La partie centrale de l'éditeur.
- **Grilles** : Lignes de battement et de mesure visuelles.
- **Encliquetage** : Activez l'encliquetage sur la grille pour un timing parfait.
- **Zoom** : Zoom avant pour un mapping détaillé ou arrière pour une vue d'ensemble.
- **Groupes intelligents (Smart Groups)** : Organisez vos pistes et vos notes avec une logique de groupage puissante.

### 3. Barre d'outils
Accès rapide aux outils de sélection, au placement des notes et aux commandes de lecture.

### 4. Actions des notes et Export
- **Actions personnalisées** : Sélectionnez n'importe quelle note pour lui attribuer une logique de jeu ou des actions spécifiques dans le panneau des propriétés.
- **Export JSON** : Exportez vos partitions dans un format JSON propre, prêt pour la production.
