# Interface : Accueil

L'écran d'accueil est votre centre de gestion de projets.

## Liste des projets
Gérez vos fichiers `.rne`. Vous pouvez ouvrir, renommer ou supprimer des projets directement d'ici.

## Sélecteur de thème
Personnalisez votre expérience avec l'icône de palette.
- **Améthyste** : Notre thème violet signature.
- **Dark/Light** : Modes classiques.
- **Chat T3** : Un thème spécial inspiré du magenta.

## Créer un nouveau projet
Entrez vos détails et importez une piste audio pour commencer.
