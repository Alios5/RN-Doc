# Guide : Gestion de projet

Gardez vos créations rythmiques organisées.

## Organisation des fichiers
Gardez vos fichiers `.rne` dans un dossier dédié. RhythmNator suit les chemins relatifs vers vos fichiers audio, il est donc préférable de garder la musique dans le même répertoire.

## Duplication de projets
Utilisez la fonction "Dupliquer" sur l'écran d'accueil pour créer différents niveaux de difficulté (Facile, Moyen, Difficile) à partir du même projet de base.

## Synchronisation Cloud
Bien que RhythmNator Editor soit une application de bureau, vous pouvez facilement synchroniser vos projets en utilisant des services comme GitHub ou Dropbox.
