# Interface : Éditeur

L'éditeur de partitions est l'endroit où vous créez les rythmes.

## Composants
- **Explorateur de forme d'onde** : Navigation et synchronisation.
- **Pistes de la chronologie** : Placez des notes sur différentes pistes.
- **Propriétés de note** : Personnalisez les comportements individuels des notes.

## Grille Avancée
- **Lignes de battement** : Lignes verticales pour le timing.
- **Encliquetage** : Verrouillez les notes sur la grille pour une précision parfaite.
- **Zoom** : Vue évolutive de l'ensemble de la piste.
