# Guide : Zone de travail

Apprenez à gérer la "Sheet" – votre partition rythmique.

## Création de notes
Cliquez sur la chronologie pour placer des notes. Utilisez différentes pistes pour différents instruments ou niveaux de difficulté.

## Longueur de note
Faites glisser l'extrémité d'une note pour créer un "Sustain" ou une "Note longue".

## Superposition
Organisez plusieurs pistes avec des groupes pour garder votre partition propre et gérable.
